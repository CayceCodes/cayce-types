# SourceLoupe-types

A TypeScript npm package.

## Installation

```bash
npm install sourceloupe-types
```

## Usage

```typescript
import { ScanRule } from 'sourceloupe-types';

```

## Development

1. Install dependencies
   ```bash
   npm install
   ```

2. Run tests
   ```bash
   npm test
   ```

3. Build the package
   ```bash
   npm run build
   ```
