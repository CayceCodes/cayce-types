# my-ts-package

A TypeScript npm package.

## Installation

```bash
npm install my-ts-package
```

## Usage

```typescript
import { greet } from 'my-ts-package';

console.log(greet('World')); // Output: Hello, World!
```

## Development

1. Install dependencies
   ```bash
   npm install
   ```

2. Run tests
   ```bash
   npm test
   ```

3. Build the package
   ```bash
   npm run build
   ```
